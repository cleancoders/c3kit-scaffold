# Security Policy

## Supported Versions

The most recent published `2.x` release on Clojars receives security fixes.

| Version | Supported          |
| ------- | ------------------ |
| 2.x     | :white_check_mark: |
| < 2.0   | :x:                |

## Reporting a Vulnerability

Please report security issues privately via [GitHub's "Report a vulnerability"](https://github.com/cleancoders/c3kit-scaffold/security/advisories/new) workflow, or by emailing the maintainer at gina@cleancoders.com. Do not open a public issue for security-sensitive reports.

We aim to acknowledge reports within 5 business days.
